# BMI Calculator

## Description
This project is a BMI (Body Mass Index) Calculator developed using Python as part of the Oasis Infobyte Python Programming Internship.

## Objective
To calculate the BMI of a user using height and weight and classify the result into different health categories.

## Technologies Used
- Python

## How It Works
1. User enters weight in kilograms.
2. User enters height in meters.
3. BMI is calculated using the formula:
   BMI = weight / (height * height)
4. The program displays the BMI value and category.
