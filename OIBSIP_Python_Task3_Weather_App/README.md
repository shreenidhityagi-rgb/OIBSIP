# Basic Weather App

## Description
This project is a Basic Weather Application developed using Python as part of the Oasis Infobyte Python Programming Internship.

## Objective
To fetch and display current weather information of a city entered by the user using a weather API.

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## How It Works
1. User enters the name of a city.
2. The program sends a request to the weather API.
3. Weather details like temperature and condition are fetched.
4. The weather information is displayed to the user.

## Features
- City-based weather search
- Displays temperature and weather condition
- Simple command-line interface
